# Ebinezer M — Personal Portfolio

> A dark editorial portfolio with a terminal hacker aesthetic, built with pure HTML, CSS & JavaScript.

🌐 **Live Site:** [ebinezer-portfolio.vercel.app](https://portfolio-nu-hazel-70.vercel.app/)

---

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Scroll reveal, custom cursor, parallax
- **Google Fonts** — Syne · Instrument Serif · DM Mono
- **Vercel** — Deployment

---

## Features

- Custom animated cursor with follower
- Scroll-triggered reveal animations
- Hero parallax effect
- Animated terminal card (About section)
- Infinite marquee (Coursework)
- Fully responsive — mobile, tablet & desktop
- Active nav link tracking on scroll
- Zero dependencies, zero frameworks

## Deployment

Deployed on **Vercel**. Any push to `main` triggers an automatic redeploy.

To deploy your own fork:
1. Fork this repo
2. Import it at [vercel.com](https://vercel.com)
3. Select **Other** as framework preset
4. Click **Deploy**
